# Bridge Hub — Marketing site

Separate Next.js app from screening. Deployed as its own Vercel project.

```bash
# From repo root
npm run dev:marketing    # http://localhost:3000
npm run build:marketing
```

Screening app: repo root, `npm run dev:screening` → http://localhost:3001

Docs: [docs/marketing/README.md](../../docs/marketing/README.md)
